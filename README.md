# Project description
This website is a modern interactive digital notice board designed for productivity, creativity, and idea organization. It combines real-time utilities like a world clock and calendar with a fully customizable workspace where users can create draggable sticky notes, sketch freely using colored markers, upload images, and arrange everything visually on a realistic virtual notice board.

Users can write notes, resize and move them anywhere, draw directly on the board, erase sketches, and pin images just like a physical creative workspace. The platform also supports world time zones, dark mode, smooth animations, responsive design, and automatic saving using local storage so all content remains even after refreshing the page.

The website creates a clean and immersive experience that feels like a blend of a digital planner, brainstorming board, and creative workspace — perfect for students, creators, teams, designers, and anyone who likes organizing ideas visually.
